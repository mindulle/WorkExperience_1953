# 1953형제돼지국밥 리뷰분석 자동화_로컬웨이브 (Notion 기획 요약)

## 1. 프로젝트 주차별 주요 기록
- **1주차**: 일경험 서류 정리 및 프로그램 이해, 오리엔테이션 종료
- **2주차 진행사항**:
  - 회의 결과 외부 노션 페이지 하나에 필요한 기능을 다 넣고 피드백 받기로 결정됨.
  - 샘플 n8n 워크플로우 작성 및 프로토타이핑.
  - **피드백(멘토링 후)**: 프로토타입 자료가 중구난방이고 BI 도구를 통으로 넣어 가시성이 떨어짐 ➔ **Google Sheet 기반으로 변경 요청 확정** (병목의 명확화 필요).
- **3주차 진행사항**:
  - 데이터 수집 및 정제 도구 구축 시작.
  - 플랫폼 특성 파악을 위해 n8n 대신 더 많은 기능을 유연하게 구현할 수 있는 **로컬 개발 환경(클로드 등)**을 채택.
  - 로컬 환경설정, 공유 폴더 및 Google Sheet 연동 성공.
  - 디자인 프로토타이핑(보류 처리됨) 대신, 팀원들과의 '클로드 데이터분석 환경설정' 이슈를 우선순위로 진행.
- **4주차**: (진행 예정/현황 체크 중)

## 2. 주요 칸반 보드 구조
- 프로젝트 현황 / 미팅 일정 / 처리전 결정사항 / 처리목표일 / 전체할일
